# AppMeuCombustivel

Aplicativo desenvolvido em .NET MAUI para calcular qual combustível compensa mais: Etanol ou Gasolina.

## Funcionalidades

- Entrada do preço do Etanol
- Entrada do preço da Gasolina
- Cálculo automático da melhor opção
- Exibição do resultado em alerta

## Regra utilizada

Se o preço do Etanol for maior que 70% do preço da Gasolina, compensa abastecer com Gasolina. Caso contrário, compensa Etanol.

## Tecnologias utilizadas

- C#
- .NET MAUI
- XAML
